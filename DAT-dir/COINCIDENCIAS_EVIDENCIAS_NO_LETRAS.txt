---------------------

1. IDENTIFICAR PALABRAS CANDIDATAS
   - Buscar palabras con elementos no alfabéticos (números, símbolos, patrones)
   - Filtrar por contexto técnico relevante
   - Priorizar por rareza y significado

2. EVALUAR COINCIDENCIAS ELEMENTO-CONCEPTO
   - Para cada elemento no alfabético, buscar conceptos técnicos asociados
   - Evaluar si el contexto del sistema relaciona ambos
   - Calcular probabilidad de coincidencia real

3. VALIDAR ESTRUCTURA DE ACRÓNIMO
   - Verificar si todos los elementos tienen significado

   - Estándares documentados del acrónimo
   - Uso consistente en el sistema
   - Referencias cruzadas válidas

5. GENERAR VEREDICTO CON NIVEL DE CONFIANZA
   - Combinar todas las evidencias
   - Calcular confianza matemática
   - Proporcionar recomendación final

function detectar_acronimo_completo(palabra, contexto_sistema):
    # Método 1: Detectar por letras raras
    resultado_letras = detectar_acronimo_por_letras_raras(palabra, contexto_sistema)
    
    # Método 2: Detectar por elementos no alfabéticos
    resultado_no_alfabeticos = detectar_acronimo_no_alfabetico(palabra, contexto_sistema)
    
    # Combinar evidencias
    confianza_combinada = max(resultado_letras.confianza, resultado_no_alfabeticos.confianza)
    
    # Evidencia combinada
    if resultado_letras.confianza > 0.8 or resultado_no_alfabeticos.confianza > 0.8:
        veredicto = "ACRÓNIMO_CONFIRMADO"
    elif resultado_letras.confianza > 0.5 or resultado_no_alfabeticos.confianza > 0.5:
        veredicto = "INVESTIGACIÓN_REQUERIDA"
    else:
        veredicto = "NO_ES_ACRÓNIMO"
    
    return {
        "palabra": palabra,
        "resultado_letras": resultado_letras,
        "resultado_no_alfabeticos": resultado_no_alfabeticos,
        "confianza_combinada": confianza_combinada,
        "veredicto_final": veredicto
    }
