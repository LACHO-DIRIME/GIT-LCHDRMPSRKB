🔍 SISTEMA DE EVALUACIÓN DE COINCIDENCIAS

MÉTODO DE APLICACIÓN:
---------------------

1. IDENTIFICAR PALABRAS CANDIDATAS
   - Buscar palabras con letras raras (K, Q, X, Z, W, J, Y)
   - Filtrar por contexto técnico relevante
   - Priorizar por frecuencia y significado

2. EVALUAR COINCIDENCIAS LETRA-CONCEPTO
   - Para cada letra rara, buscar conceptos técnicos asociados
   - Evaluar si el contexto del sistema relaciona ambos
   - Calcular probabilidad de coincidencia real

3. VALIDAR ESTRUCTURA DE ACRÓNIMO
   - Verificar si todas las letras tienen significado
   - Evaluar coherencia semántica del conjunto
   - Confirmar lógica interna del acrónimo

4. BUSCAR EVIDENCIAS FUERTES
   - Documentación explícita del acrónimo
   - Uso consistente en el sistema
   - Referencias cruzadas válidas

5. GENERAR VEREDICTO CON NIVEL DE CONFIANZA
   - Combinar todas las evidencias
   - Calcular confianza matemática
   - Proporcionar recomendación final

function detectar_acronimo(palabra, contexto_sistema):
    # Paso 1: Identificar letras raras
    letras_raras = identificar_letras_raras(palabra)
    
    if not letras_raras:
        return {"veredicto": "NO_ES_ACRONIMO", "confianza": 0.1, "razon": "sin_letras_raras"}
    
    # Paso 2: Evaluar coincidencias
    coincidencias = evaluar_coincidencias_conceptuales(letras_raras, contexto_sistema)
    
    # Paso 3: Validar estructura
    estructura_valida = validar_estructura_acronimo(palabra, coincidencias)
    
    # Paso 4: Buscar documentación
    documentacion = buscar_documentacion_explicita(palabra)
    
    # Paso 5: Calcular confianza
    confianza = calcular_confianza(letras_raras, coincidencias, estructura_valida, documentacion)
    
    # Paso 6: Generar veredicto
    veredicto = generar_veredicto(confianza)
    
    return {
        "palabra": palabra,
        "letras_raras": letras_raras,
        "coincidencias": coincidencias,
        "estructura_valida": estructura_valida,
        "documentacion": documentacion,
        "confianza": confianza,
        "veredicto": veredicto
    }
