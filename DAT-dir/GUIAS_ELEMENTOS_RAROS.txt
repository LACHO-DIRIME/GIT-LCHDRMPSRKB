GUÍAS Y ELEMENTOS POCO FRECUENTES - SISTEMA DE DETECCIÓN
===========================================================

🔍 GUÍA PRINCIPAL - DETECCIÓN DE ACRÓNIMOS POR LETRAS RARAS:

PASO 1: IDENTIFICACIÓN DE LETRAS RARAS
---------------------------------------
Letras de alta prioridad (más raras y significativas):
- K, Q, X, Z, W, J, Y

Criterio de rareza:
- Frecuencia < 1% en vocabulario técnico general
- Uso específico en contextos tecnológicos
- Asociación fuerte con conceptos técnicos

PASO 2: ANÁLISIS DE COINCIDENCIAS
---------------------------------
Para cada letra rara identificada:

1. Buscar conceptos técnicos que inicien con esa letra
2. Verificar si el contexto del sistema relaciona ambos
3. Evaluar la coherencia semántica
4. Comprobar si el resto de letras tienen significado

PASO 3: VALIDACIÓN CONTEXTUAL
------------------------------
- Buscar documentación explícita
- Verificar uso consistente en el sistema
- Confirmar coherencia con otros componentes
- Validar lógica interna del acrónimo

🔍 MÉTODO DE EVALUACIÓN DE COINCIDENCIAS:

NIVELES DE EVIDENCIA:
---------------------

Nivel 1 - EVIDENCIA FUERTE:
- Letra rara + coincidencia conceptual directa
- Documentación explícita del acrónimo
- Uso consistente en el sistema
- Coherencia semántica completa

Ejemplo: KALIL (K = Kernel, documentado, usado consistentemente)

Nivel 2 - EVIDENCIA MODERADA:
- Letra rara + coincidencia conceptual probable
- Uso contextual pero sin documentación explícita
- Coherencia semántica parcial

Ejemplo: Palabras con Q, X, W sin confirmación

Nivel 3 - EVIDENCIA DÉBIL:
- Letra rara sin coincidencia conceptual clara
- Uso ambiguo o múltiple
- Sin coherencia semántica evidente

Ejemplo: Palabras con Z sin contexto claro

🔍 ALGORITMO DE DETECCIÓN:

FUNCIÓN PRINCIPAL:
------------------

detectar_acronimo_por_letras_raras(palabra):
    1. Identificar letras raras en la palabra
    2. Para cada letra rara:
        a. Buscar conceptos técnicos asociados
        b. Evaluar coincidencia con contexto
        c. Calcular probabilidad de acrónimo
    3. Combinar evidencias de todas las letras
    4. Generar veredicto con nivel de confianza
    5. Proporcionar recomendación de investigación

VARIABLES DE EVALUACIÓN:
-----------------------

- Rareza_letra: 0.0 - 1.0 (más rara = más valor)
- Coincidencia_conceptual: 0.0 - 1.0 (más coincidencia = más probabilidad)
- Coherencia_semantica: 0.0 - 1.0 (más coherencia = más confianza)
- Documentacion_explicita: 0.0 - 1.0 (documentación = confirmación)

PESOS DE EVALUACIÓN:
------------------

- Rareza_letra: 30%
- Coincidencia_conceptual: 40%
- Coherencia_semantica: 20%
- Documentacion_explicita: 10%
