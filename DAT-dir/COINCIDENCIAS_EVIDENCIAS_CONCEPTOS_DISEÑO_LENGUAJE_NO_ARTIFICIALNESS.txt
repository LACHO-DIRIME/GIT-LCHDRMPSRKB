MÉTODO DE APLICACIÓN:
---------------------

1. IDENTIFICAR INDIVIDUALIDAD FUNDAMENTAL EN DISEÑO
   - Buscar individualidad sintáctica básica
   - Identificar autonomía de paradigma
   - Detectar identidad de lenguaje
   - Encontrar propiedad de abstracción

2. EVALUAR COINCIDENCIAS INDIVIDUALIDAD-AUTONOMÍA EN DISEÑO
   - Para cada individualidad fundamental, buscar aplicación específica en diseño
   - Evaluar si el contexto del sistema relaciona ambos
   - Calcular probabilidad de concepto de diseño único
   - Validar coherencia de individualidad y autonomía en diseño

3. VALIDAR ESTRUCTURA DE CONCEPTO DE DISEÑO
   - Verificar si cada individualidad tiene aplicación en diseño
   - Evaluar coherencia del conjunto
   - Confirmar lógica interna del concepto
   - Comprobar aplicación de individualidad en diseño

4. BUSCAR EVIDENCIAS FUERTES EN DISEÑO
   - Aplicación de individualidad del concepto en diseño
   - Uso consistente en el sistema de diseño
   - Referencias cruzadas válidas en diseño
   - Documentación de individualidad en diseño

5. GENERAR VEREDICTO CON NIVEL DE CONFIANZA
   - Combinar todas las evidencias de diseño
   - Calcular confianza matemática
   - Proporcionar recomendación final

function detectar_concepto_diseño(palabra, contexto_sistema):
    # Paso 1: Identificar individualidad fundamental en diseño
    individualidad_fundamental = identificar_individualidad_fundamental_diseño(palabra)
    
    if not individualidad_fundamental:
        return {"veredicto": "NO_ES_CONCEPTO", "confianza": 0.1, "razon": "sin_individualidad_fundamental_diseño"}
    
    # Paso 2: Evaluar coincidencias
    coincidencias = evaluar_coincidencias_diseño(individualidad_fundamental, contexto_sistema)
    
    # Paso 3: Validar estructura
    estructura_valida = validar_estructura_diseño(palabra, coincidencias)
    
    # Paso 4: Buscar aplicación de individualidad en diseño
    aplicacion = buscar_aplicacion_diseño(palabra)
    
    # Paso 5: Calcular confianza
    confianza = calcular_confianza_diseño(individualidad_fundamental, coincidencias, estructura_valida, aplicacion)
    
    # Paso 6: Generar veredicto
    veredicto = generar_veredicto_diseño(confianza)
    
    return {
        "palabra": palabra,
        "individualidad_fundamental": individualidad_fundamental,
        "coincidencias": coincidencias,
        "estructura_valida": estructura_valida,
        "aplicacion": aplicacion,
        "confianza": confianza,
        "veredicto": veredicto
    }

    # Método 2: Detectar por diseño semántico

    # Método 3: Detectar por diseño estructural

    # Método 5: Detectar por diseño de identidad
    resultado_identidad = detectar_concepto_diseño_identidad(palabra, contexto_sistema)
    
    # Método 6: Detectar por diseño de abstracción
    resultado_abstraccion = detectar_concepto_diseño_abstraccion(palabra, contexto_sistema)
    
    # Combinar evidencias
    confianza_combinada = max(
        resultado_sintactico.confianza, 
        resultado_semantico.confianza, 
        resultado_estructural.confianza,
        resultado_paradigmas.confianza,
        resultado_identidad.confianza,
        resultado_abstraccion.confianza
    )

        veredicto = "INVESTIGACIÓN_REQUERIDA"

        veredicto = "NO_ES_CONCEPTO"

        "palabra": palabra,
