### 1.1 DEFINICIÓN Y PROPÓSITO

**Definición de Masking Directory:**
Masking Directory es el sistema de ocultamiento y protección del ecosistema LACHO, diseñado como la implementación tecnológica del icono `masking folder` que gestiona el acceso, la privacidad y la seguridad de la información mediante protocolos de enmascaramiento dinámico.

**Doble Naturaleza de MASKING:**
1. **{masking} como módulo de SOCIAL BIBLIOTECA:** H52 (Lo Mantenido, La Montaña), $thu (jueves), tipo (M) Masivo/Estable
2. **`masking folder` como iconografía:** H23 (La Desintegración, La Montaña) - Poda ontológica, GREEN-KNOWLEDGE

**Relación Hexagramática Triple:**
- **H52 (SOCIAL):** Principio de quietud y protección mantenida
- **H23 (Iconografía):** Principio de desintegración y poda necesaria
- **STACKING:** Ambos operan en los pilares de La Montaña
- **SAMU:** $thu coordina la auditoría de enmascaramiento

**Declaración de Vinculación:**
```
MASKING_DIRECTORY → {masking} (SOCIAL) : protección y ocultamiento
MASKING_DIRECTORY → H52 (STACKING) : pilar de montaña protector
MASKING_DIRECTORY → H23 (iconografía) : poda ontológica necesaria
MASKING_DIRECTORY → $thu (SAMU) : auditoría los jueves
```

Ambas están intrínsecamente asociadas: el módulo SOCIAL implementa la funcionalidad descrita por la iconografía.

**Propósito Fundamental (según LACHO):**
- **Ser el sistema de ocultamiento y poda** del ecosistema
- **Proteger información sensible** mediante enmascaramiento
- **Gestionar acceso selectivo** a recursos del sistema
- **Aplicar test delta-Ψ** para permanencia o caída de máscara
- **Mantener equilibrio crecimiento/poda** del sistema

**Protocolo Delta-Ψ:**
```

1. Calcular delta-Ψ del recurso
2. Evaluar condición ΔΨ ≤ 0 o ΔΨ > 0
3. Determinar permanencia de máscara
4. Si ΔΨ ≤ 0 → máscara permanente (purga)
5. Si ΔΨ > 0 → máscara cae, contenido fluye
6. Cambiar estado visual del icono
7. Mantener equilibrio crecimiento/poda
8. Reportar estado a HEADCAT
```

**Niveles de Delta-Ψ:**
- **ΔΨ ≤ 0**: Máscara permanente, purga activa
- **ΔΨ > 0**: Máscara cae, contenido fluye
- **ΔΨ ≈ 0**: Estado límite, evaluación continua
- **ΔΨ >> 0**: Flujo máximo, acceso completo
