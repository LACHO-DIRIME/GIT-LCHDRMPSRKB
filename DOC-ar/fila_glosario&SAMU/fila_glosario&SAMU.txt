──────────────────────────────────────────────────────────────────────────────
Términos: 489 | Letras A–Z + UF[H01-H02]
Columnas:
  TÉRMINO          → nombre canónico del glosario
  FILE             → archivo de origen (letra)
  GLOSARIO_N       → nivel del glosario (N1 fundamento · N2 módulo · N3 estado · N4 ref)
  CLASE            → Clase: declarada en el archivo
  TAX_LEVEL        → nivel taxonomía SAMU (0=constitucional · 1=biblioteca · 2=módulo · 3=operador)
  SCALAR_MIN       → Scalar S mínimo requerido para operar con este término
  TAX_CLASS        → clase interna de taxonomy.py
  SAMU_ACTION      → acción que SAMU @ ejecuta al encontrar este término
  DESCRIPCIÓN      → primeras líneas de definición del glosario
  VER              → referencias cruzadas del glosario
──────────────────────────────────────────────────────────────────────────────

LEYENDA DE ACCIONES SAMU @:
  TAX 0 · S≥0.95 · constitucional → verifica invariante antes de cualquier operación
  TAX 1 · S≥0.88 · biblioteca     → audita jurisdicción · coordina con biblioteca
  TAX 2 · S≥0.85 · módulo/proto   → activa/monitorea módulo · audita salida
  TAX 3 · S≥0.80 · operador/ref   → registra en RAG · ajusta Scalar S · clasifica

RAG SIGNAL FORMAT por término:
  [RAG: SAMU :: {TÉRMINO} :: {GLOSARIO_N} :: {TAX_CLASS}]
